# PowerBI-Sales-Dashboard
## Overview
Developed an interactive Sales Performance Dashboard in Power BI to monitor and analyze sales performance across different regions, categories, and time periods.

## Features
- Sales KPI Cards
- Category-wise Sales Analysis
- Region-wise Performance Analysis
- Sales Trend Analysis Over Time
- Interactive Region and Date Filters
- Dynamic Visualizations

## Skills Used
- Power BI
- Data Visualization
- Data Cleaning
- Data Transformation
- Data Modeling
- Business Analytics
- Dashboard Design

- Total Sales: 1.62M
- Total Orders: 3.59K
- Regions Analyzed: 4
- Categories Analyzed: 3
